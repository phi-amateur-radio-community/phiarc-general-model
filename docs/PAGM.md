# PhiArc General Model (PAGM)

The PAGM is a model header file that contains human-readable model information and an index of parameter files.

## File Structure

### Model Configuration Information

> This part is read by the program.

This section contains information about the model configuration, such as numbers of layers in the Encoder and Decoder.

### Index of Parameter Files

> This part is read by the program.

This section contains an index of parameter files.

### Other Information

> This part will not be read by the program

This section contains other information that is not read by the program, such as the model description and copyright information.
